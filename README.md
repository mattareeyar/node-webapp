# How to install

Use powershell or cmd and type by order, please see below.

- `git clone https://github.com/mattareeyar/node-webapp.git`
- `cd node-webapp`
- > install package dependency in this project.
  - `npm install`

# How to test local

- > run main js file.
  - `npm run serve`
